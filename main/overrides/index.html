Dashboards

Dashboards are used to add metrics, logs and traces to an application. They can be customized via plugins.

Extensible via Plugins

kobs can be extended via plugins. kobs is shipped with a wide range of built-in plugins like, e.g. Prometheus, Elasticsearch, Jaeger and Opsgenie.

Applications

Applications can be used to improve the observability of your Kubernetes and Cloud workloads. Connect your Kubernetes / Cloud workloads with the corresponding metrics, logs and traces.

Metrics

Explore your applications metrics via the built-in Prometheus plugin.

Logs

View and filter the logs of your applications via the Elasticsearch or klogs plugins.

Traces

Access and compare the traces of your applications, via the Jaeger plugin.
